# swiper: a high performance password cracker

# EXAMPLE

```console
$ bench-swiper
latency (ns)    throughput (password/sec)
5.37            1.86e+08
```

# ABOUT

swiper is a library for regenerating passwords from Cisco IOS^tm type 7 encrypted configurations.
